# 🧶 The Crochet Charm

A responsive crochet business website built using Django and Bootstrap.

## ✨ Features

* Responsive Design
* Product Management through Django Admin
* Product Search Functionality
* WhatsApp Order Integration
* Contact Form
* Email Notifications
* Custom Order Section
* Instagram Integration
* Bootstrap UI

## 🛠️ Tech Stack

* Python
* Django
* Bootstrap 5
* SQLite
* HTML
* CSS
* JavaScript

## 📸 Website Sections

* Home
* Products
* About Us
* Why Choose Us
* Custom Orders
* Contact Us

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/khushi9956/The-Crochet-Charm.git
```

Navigate to project folder:

```bash
cd The-Crochet-Charm
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run migrations:

```bash
python manage.py migrate
```

Start server:

```bash
python manage.py runserver
```

Open:

```text
http://127.0.0.1:8000/
```

## 💖 About

The Crochet Charm is a handmade crochet business website created to showcase and manage beautiful crochet products such as bouquets, hair accessories, keychains, bags, and custom handmade gifts.
